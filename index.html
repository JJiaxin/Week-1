Static Website

My example of a static website would be Wikipedia.
Wikipedia provides the user with the information they request. You click on a link, it opens and there is no longer interaction between the computer and the server until the user clicks again.

Dynamic Website

An example of a dynamic website would be Netflix.
Based on the user's watch history and likes, the page would constantly update itself featuring where the user left off on a certain show and what the user might like.
